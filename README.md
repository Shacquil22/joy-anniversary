# Joy — 2 Year Anniversary

A personal anniversary page for Joy.

## Live link

**https://shacquil22.github.io/joy-anniversary/**

## Setup GitHub Pages

1. Open [Settings → Pages](https://github.com/Shacquil22/joy-anniversary/settings/pages)
2. Source: **Deploy from a branch**
3. Branch: **main**, folder: **/** (root)
4. Save

## Add photos

Upload images to the `photos/` folder as:

- `01.jpg` — Date night smiles
- `02.jpg` — Couch cuddles
- `03.jpg` — Golden hour together
- … up to `20.jpg` — Forever us

Or use the **"Add Your Photos Here"** button on the page to load them from your device.
